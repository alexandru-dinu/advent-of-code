t = 13 * k2 - 1 => t ≡ 12 (mod 13)
t = 59 * k3 - 4 => t ≡ 55 (mod 59)
t = 31 * k4 - 6 => t ≡ 25 (mod 31)
t = 19 * k5 - 7 => t ≡ 12 (mod 19)

All moduli are pairwise coprime. We can apply CRT.
